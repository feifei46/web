# 淄博烧烤爆红路径解码 · 文旅目的地运营工具箱
> 基于公开数据的文旅目的地热度分析与可视化 —— 大学生数据要素素质大赛参赛作品
本仓库为参赛作品的可视化展示平台，提供三个可直接使用的文旅目的地运营诊断工具。

## 工具箱一览
| 工具 | 功能 | 适用场景 |
|------|------|----------|
| **情感预警表** | 实时监测目的地舆情，负面情绪超阈值自动预警 | 日常舆情监控 |
| **关键词看板** | 提取正负面高频关键词，追踪语义漂移 | 洞察用户关注焦点 |
| **热度指数卡** | 从五个维度量化评估目的地的爆红潜力 | 项目启动前潜力评估 |

## 工具详情

### 1. 情感预警表
基于社交媒体评论的情感分析结果，设定两级预警阈值：
| 阈值 | 预警等级 | 建议动作 |
|------|----------|----------|
| 负面情绪占比 > 30% | 黄色预警 | 启动舆情监测，准备应对方案 |
| 负面情绪占比 > 50% | 红色预警 | 暂停营销投放，主动回应，排查问题 |
| “排队”关键词周环比 > 50% | 黄色预警 | 评估承载能力，发布分流引导 |
| “贵/涨价”关键词周环比 > 30% | 黄色预警 | 检查定价策略，发布优惠说明 |

### 2. 关键词看板
定期追踪正负面关键词TOP20及排名变化，用于：
- 识别核心卖点，强化传播内容
- 快速发现槽点，指导整改方向
- 评估传播策略效果，捕捉“语义漂移”

### 3. 热度指数卡
从五个维度对目的地爆红潜力进行量化评估（满分100分）：
| 评估维度 | 权重 | 评分内容 |
|----------|------|----------|
| 话题性 | 25% | 社交媒体讨论量、热搜潜力、内容自传播力 |
| 承接力 | 25% | 政府响应速度、基础设施配套、服务体系 |
| 独特性 | 20% | 核心体验差异化程度、不可替代性、文化内涵 |
| 传播力 | 15% | UGC内容数量、用户自发分享意愿、平台合作基础 |
| 持续力 | 15% | 复购可能性、口碑稳定性、产业延展空间 |
